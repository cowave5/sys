## sys

示例：通用管理系统

- sys-ui      前端界面
- sys-admin   系统管理
- sys-code    代码生成
- sys-blog    博客管理
- sys-flow    流程管理
- sys-eureka  服务注册（已弃用）
- sys-gateway 服务网关
- sys-model   模型定义
- sys-quartz  定时任务
